## Code Examples

The `code/` directory contains practical code snippets and examples for each topic:

### interpreted

This folder contains code snippets associated with the Python interpreter, including bytecode examination with the `dis` module.

### namespaces

This folder contains code snippets to illustrate variables and namespaces in Python, demonstrating scope, mutable/immutable arguments, and variable behavior.

### object_oriented

This folder contains code snippets associated with Python object-oriented programming, illustrating classes, objects, mutability, and object variables.

### garbage_collector

This folder contains code snippets associated with the Python garbage collector, demonstrating reference counting, circular references, and best practices.

### imports

This folder contains code snippets associated with Python imports, including module creation and import statements.

### copies

This folder contains code snippets demonstrating shallow copies, deep copies, and copying behavior for different data structures.

## Getting Started

1. Read through the learning sections in order to understand fundamental concepts
2. Explore the corresponding code examples in the `code/` directory
3. Run the code examples to see concepts in action
4. Experiment by modifying and extending the examples
